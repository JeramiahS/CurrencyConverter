# CurrencyConverter

## About This Project

CurrencyConverter converts a given value of one country's currency into another country's currency using a real-time exchange rate. It currently supports 170 different countries to convert between.

## How to Use CurrencyConverter

1. Enter the amount of money to convert. It must be a non-negative, non-zero number.
2. Select the currency code of the country you want to convert from within the dropdown menu.
3. Select the currency code of the country to want to convert to within the dropdown menu.
4. Click on the 'Convert' button. The conversion result will be displayed in the area below the button.
